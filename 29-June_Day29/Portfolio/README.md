# 3D Portfolio Website

A stunning 3D book-style portfolio website developed using HTML, CSS, and vanilla JavaScript. 

## Features
- **3D Book Effect:** Pages turn smoothly giving a realistic physical book feel.
- **Scroll Navigation:** Seamlessly navigate between pages using mouse scrolling.
- **Responsive Animations:** Elements animate flawlessly on page load and during interaction.
- **Modern UI:** Intentionally designed interfaces utilizing boxicons for iconography.

## Getting Started
To view this project locally:
1. Clone the repository.
2. Open `index.html` in any modern web browser.

## Technologies Used
- HTML5
- CSS3 (Custom animations, 3D Transforms)
- JavaScript (DOM manipulation, event listeners)
- Boxicons
